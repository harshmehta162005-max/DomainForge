# Firecrawl (Web Research)

**Tags:** [Scraping, Research, Data Extraction]

When you need up-to-date information from websites:
- Use Firecrawl MCP to scrape and convert pages into clean structured markdown/JSON.
- Save results to files instead of dumping into context.
- Use for competitor analysis, pricing research, documentation, etc.

Especially useful for researching domain tools, registrar APIs, and current market data.
