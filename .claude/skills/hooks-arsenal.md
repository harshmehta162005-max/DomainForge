# Hooks Arsenal

**Tags:** [Hooks, Automation, Quality, CI/CD]

Use lifecycle hooks to automate quality:
- **PostToolUse**: Auto-format code after every Write/Edit.
- **PreToolUse**: Block dangerous writes (e.g. .env, migrations).
- **SessionStart**: Inject context, Caveman mode, or project memory.

Configure these in `.claude/settings.json`. This is one of the most powerful production setups.
