# Session Handoff

**Tags:** [Context Transfer, Handoff, Multi-Agent]

At the end of a session or when switching agents, create a professional HANDOFF.md file.

Content should include:
- Current state of the project
- Key decisions made
- Open questions
- Next priorities
- Important context the next agent must know

Write it as if you are a CTO handing off to a senior developer. Make it clear and actionable.
